Usage guide for <admin.exe>:

[GREEN] 1. read-all
[NORMAL]   - Description: Retrieves and displays all current sensor readings.
[RED]      - Example: Enter 'read-all' to see all sensor data.

[GREEN] 2. set-temperature
[NORMAL]    - Description: Sets the temperature for the sensor.
[RED]       - Example: Enter 'set_temperature 23.0'
[NORMAL]    - Input: A floating-point number representing the temperature.

[GREEN] 3. export
[NORMAL]    - Description: Exports the sensor data to a specified directory.
[RED]       - Example: Enter 'export C:/Documents/Sensor-data'
[NORMAL]    - Input: A valid directory path where the output file will be saved.

[GREEN] 4. exit
[NORMAL]    - Description: Exits the console application.
[RED]       - Example: Enter 'exit' to close the application.

[NORMAL] If you encounter an error, please ensure you have entered a valid command as described above.
